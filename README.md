# EA Creatives - Landing

Landing page construida con React, TypeScript, Vite y Tailwind CSS.

## Requisitos

- Node.js 18+
- npm 9+

## Desarrollo local

```sh
npm install
npm run dev
```

La app queda disponible en `http://localhost:8080`.

## Build de producción

```sh
npm run build
npm run preview
```

## Testing y lint

```sh
npm run test
npm run lint
```

## Stack principal

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Framer Motion
